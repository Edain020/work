作品名:記憶の砂時計
制作期間:2021年2月～2021年5月
制作人数:15人(サウンド、ムービー制作含む)

概要
日本ゲーム大賞2021アマチュア部門に応募するため制作した作品です。
ゲームのメインとなる巻き戻しの処理、プレイヤーキャラクターのアニメーションの実装、ポーズメニュー、フェードイン・フェードアウトの実装、チュートリアルの表示等を担当しました。

アピールポイント
一定時間で起こる巻き戻しのシステム、アニメーションの管理に力を入れて制作しました


操作(キーボード)
矢印キー:移動
Enter:決定
BackSpace:戻る
P:ポーズメニュー
Space:ジャンプ
E:物をつかむ
